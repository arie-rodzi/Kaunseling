# JKM Psychological Service Analytics Dashboard

## Overview

This prototype dashboard was developed as an early analytical prototype for the tender:

> **Kajian Penilaian Keberkesanan Terhadap Perkhidmatan Psikologi dan Kaunseling di Jabatan Kebajikan Masyarakat (JKM)**

The system demonstrates a data-driven dashboard concept for monitoring, evaluating and analysing psychological and counselling services performance.

This prototype is developed using **Streamlit** with simulated data for demonstration purposes only.

---

# Main Features

## Executive Dashboard
- Overall Effectiveness Score
- Satisfaction Index
- Intervention Success Rate
- Total Cases Monitoring
- High Risk Case Detection

## State Analytics
- State performance comparison
- Effectiveness ranking
- Regional performance analysis

## Intervention Analytics
- Intervention type analysis
- Success distribution
- Service effectiveness trends

## Psychosocial Indicators
- Emotional improvement
- Stress reduction
- Recovery readiness
- Behavioural outcome monitoring

## Interactive Visual Analytics
- KPI cards
- Interactive charts
- Trend analysis
- Comparative analytics
